El Producto de un escalar por un vector

Sea U = (x, y) un vector y "k" un escalar

Formula

K · U = (K · X, K · Y)

Ejemplo

Sea S = (2, -7) y k = 3

k·S = (3·2, 3·(-7))

k·S = (6, -21)

Producto Punto o escalar

Nota:

  • Si el producto escalar es = 0 se dice que los vectores son perpendiculares u ortogonales.

  • Si el producto punto es menor a 0 el angulo entre los dos vectores debe ser entre 90° y 179°.
